# Sửa dữ liệu kết nối thu thập

1. Vào phân hệ **THU THẬP DỮ LIỆU** -\> **Chuyển đổi dữ liệu thu thập**
2. **Sửa** dữ liệu trên màn hình thêm/sửa
3. Nhấn **Lưu** để hoàn thành
